Task 4: Using regex to parse country-level activity from the MMSI field (30 points)

An AIS transmission consists of 17 columns, but the MMSI is the most important field. The Maritime Mobile Service Identity (MMSI) is a unique nine digit identifier to identify a ship or a coast radio station. The MMSI follows an explicit naming convention that allows us to ascertain great detail about each indentifier. To see the standard format, view the MMSI_Format_Overview.pdf document provided in the lab zip file or access a version directly at the following website. Of particular interest, we see that the first three characters of an individual ships MMSI signifies the Maritime Identification Digits (MID). The MID is the national entity responsible for issuing/managing the MMSI for that particular ship. You can access a lookup table for all MID values here. For example, an MMSI of 503123456 would signify a ship associated with Australia. We will assume the MID to be a direct proxy of a ships national origin.

For Task 4, you will determine the total unique vessel count of ships that have a national origin from the following nations: Canada, Singapore, and Cayman Islands. To accomplish this task, you will use some form of regex operations in the programming language of your choice (i.e.python, r or bash). Present the code to accomplish the task and a narrative describing your methods and results.

MMSI Codes * Canada - 316 * Singapore - 563, 564, 565, 566 * Cayman Islands - 319

# Reading in data
AIS_2018 <- read.csv("AIS_2018_04_01.csv")

tot_vess <- length(AIS_2018[grep("^316|^563|^564|^565|^566|^319",AIS_2018$MMSI),"MMSI"])
tot_uni.vess <- length(unique(AIS_2018[grep("^316|^563|^564|^565|^566|^319",AIS_2018$MMSI),"MMSI"]))

print(paste("Out of the", tot_vess, "vessels associated with Canada, Singapore, and the Cayman Islands, there are only",  tot_uni.vess, "unique vessels"))
